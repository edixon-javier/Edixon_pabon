Etiquetas semanticas

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido. Por ejemplo, la etiqueta SECTION nos dice que contiene una sección o capítulo dentro de la página. Frente a las etiquetas semánticas tenemos otros tipos de etiquetas como las que afectan al formato, como UL, a multimedia, como VIDEO, etc.

listas en HTML5

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.

Los tipos de listas en HTML son los siguientes:

  • Listas ordenadas
  • Listas desordenadas
  • Listas de definiciones

www.manualweb.net

Enlaces en HTML5

los enlaces tambine son conocidos como links y cumplen un objetivo principal que es el de conectar una pagina web con otra. tienen un atributo obligatorio href, donde se especifica la ruta del recurso que queremos obtener. tambien tiene el atributo target, que configura como queremos visualizar el recurso o sitio que solicitamos. existen dos tipos:

  • rutas absolutas:

    • tienen unu protocolo, http o https y son unicas en la red se suelen utilizar para rutas externas.

  • rutas relativas:

    • pueden ser relaticas al punto donde nos encontramos o relativas a la raiz del proyecto.
